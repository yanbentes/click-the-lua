# Clicker

Click the moon. Made with **[Lua](https://www.lua.org/)** and **[Love2D](https://love2d.org/)** version 11.3.

<img src="click.gif" alt="click the moon" width="400"/>

## Run

` love clicker-lua/ `

- "escape" key quit
- "r" key restart
